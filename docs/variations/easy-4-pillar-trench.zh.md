# 橙4 + 蓝多挡板

![橙4 + 蓝多挡板](../images/variations/easy-4-pillar-trench.jpg)

## 概述

[橙4](../rolls/easy-4.md#橙轮) + [蓝多挡板](../rolls/pillar-trench.md)开头和中间都很简单，橙轮开头可以连续跳4个坑，蓝轮中间可以连续跳3个坑。结尾部分在玩家数量较多时有一定难度，特别是到了后期。

## 路线

第9~12圈结尾处有两种切板方法，我建议你选择玩家数量较少的路线来避免拥挤。
