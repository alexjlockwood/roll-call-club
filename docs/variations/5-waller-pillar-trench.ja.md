# 壁まし + 壁まし

![5 Waller + Pillar Trench](../images/variations/5-waller-pillar-trench.jpg)

## Overview

[壁まし](../rolls/5-waller.md) + [壁まし](../rolls/pillar-trench.md) は、オレンジ壁ましの中で最も簡単なパターンです。各周回のスタートでは衝突を避けるのが難しいですが、中盤と終盤はまっすぐで、青の中盤には広いスペースがあります。
