# 橙4 + 蓝天坑

![橙4 + 蓝天坑](../images/variations/easy-4-grand-canyon.jpg)

## 概述

[橙4](../rolls/easy-4.md#橙轮) + [蓝天坑](../rolls/grand-canyon.md)是最简单的包含蓝天坑的版型，因为橙轮开头连续4个小坑可以让你完全避开天坑。不过后期结尾处的拥挤使得改版型仍有一定难度。

## 路线

第9~12圈结尾处一共有3条路线可以选择，我建议你选择玩家数量最少的路线。
