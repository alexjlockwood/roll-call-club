# 周回数を数える

## 概要

「周回」とは、ロールが360度回転することを意味します。5分間でロールは12回転するので、12周することで耐久となります。

## 周回タイム

| 周回数 | スタート | 周回にかかる時間 |
| ----- | ---------- | -------- |
| 1     | 0:00       | 0:47     |
| 2     | 0:47       | 0:32     |
| 3     | 1:16       | 0:29     |
| 4     | 1:41       | 0:25     |
| 5     | 2:05       | 0:24     |
| 6     | 2:28       | 0:23     |
| 7     | 2:51       | 0:23     |
| 8     | 3:12       | 0:22     |
| 9     | 3:33       | 0:21     |
| 10    | 3:54       | 0:21     |
| 11    | 4:15       | 0:21     |
| 12    | 4:36       | 0:21     |

## 便利な小ネタ

ファイナルの2回目のループが始まるタイミングが7周目のスタートなので、Fall GuysのBGMをオンにしているなら簡単に7周目を判断できます。

9周目にはスライムの高さが最大になるので、最後の4周が最も難しくなります。
